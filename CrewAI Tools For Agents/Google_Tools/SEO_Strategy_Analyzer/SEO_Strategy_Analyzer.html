SEO Strategy Analyzer Usage Guide for CrewAI

This document provides detailed instructions on how to use the SEO Strategy Analyzer within the CrewAI framework. The tool is designed to analyze keyword trends, competition levels, and changes over time to optimize SEO strategies.

Prerequisites

  • Python 3.6 or newer installed on your system.
  • Access to the CrewAI framework and basic familiarity with its operation.
  • Pandas library installed for data manipulation. Install via pip: pip install pandas
  • Knowledge of SEO metrics and strategies.

Setup

Ensure the SEO Strategy Analyzer script is correctly placed within your CrewAI project's directory. Adjust the tool's parameters as needed to match your project's directory structure and data format.

Using the Tool

  1. Prepare Your Data: Organize your Excel files with keyword data in the specified source directory. Ensure each file follows the structure required by the tool.
  2. Specify Directories and Date Range: Set the source and output directories, along with the analysis date range, according to your SEO analysis needs.
  3. Run the Script: Execute the tool using Python. The tool processes each Excel file to provide insights on keyword trends, competition, and more.
  4. Analyze the Results: Review the JSON files generated in the output directory. Each file contains detailed SEO strategy analysis for the corresponding Excel file's keywords.

Example Command

# Example command to run the SEO Strategy Analyzer
python seo_strategy_analyzer.py --source_directory "/path/to/excel/files" --output_base_directory "/path/to/output" --analysis_date_range "2023-01-01 to 2023-12-31"

Customization

The SEO Strategy Analyzer can be customized to suit different Excel file formats or specific SEO analysis requirements. Adjust the analyze_keywords function in the script to refine the analysis based on your unique needs.

Troubleshooting

If you encounter any issues, verify that the source and output directories exist and that the Excel files are properly formatted. Console error messages can provide additional insight into any problems encountered during execution.
