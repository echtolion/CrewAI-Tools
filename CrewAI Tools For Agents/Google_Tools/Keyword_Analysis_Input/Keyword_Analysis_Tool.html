Keyword Analysis Tool Usage Guide for CrewAI

This document provides a comprehensive guide on how to use the Keyword Analysis Tool within the CrewAI framework. This tool automates the process of keyword research, identifying high-volume, low-competition keywords from Excel files.

Prerequisites

  • Python 3.6 or newer installed on your system.
  • Access to the CrewAI framework and understanding of its basic functionality.
  • Pandas library installed for data manipulation. Install via pip: pip install pandas
  • Basic understanding of working with Excel files and JSON data structures.

Setup

Ensure the Keyword Analysis Tool script is placed in an accessible directory within your CrewAI project. Modify the tool's parameters as necessary to fit your project's directory structure and data format.

Using the Tool

  1. Configure Source and Output Directories: Specify the source directory containing your Excel files and the output directory where JSON analysis results will be saved.
  2. Execute the Script: Run the script using Python. The tool will automatically process each Excel file, identify relevant keywords, and save the findings in a structured JSON format.
  3. Review Results: Check the output directory for JSON files named after the original Excel files. Each JSON file contains an array of high-volume, low-competition keywords and their metrics.

Example

        
        # Example command to run the Keyword Analysis Tool
        python keyword_analysis_tool.py --source_directory "/path/to/excel/files" --output_base_directory "/path/to/output"

Customization

The Keyword Analysis Tool can be customized to handle different Excel file formats or to adjust the criteria for selecting keywords. Modify the process_excel_file function within the script to fit your specific requirements.

Troubleshooting

If you encounter issues while using the tool, ensure that the source and output directories exist and that Excel files are correctly formatted. Check for error messages in the console for further diagnostics.
