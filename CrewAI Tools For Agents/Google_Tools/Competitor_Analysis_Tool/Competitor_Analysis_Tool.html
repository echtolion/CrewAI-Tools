Competitor Analysis Tool Usage Guide for CrewAI

This document outlines the steps for using the Competitor Analysis Tool within the CrewAI framework. The tool is designed to compare your website's keyword performance against that of your competitors, utilizing keyword data from Excel files.

Workflow Overview

  1. Read Excel Files: The tool iterates through each Excel file in the specified directory, extracting keyword performance data for your website and competitors.
  2. Extract Keyword Performance Data: Data including average monthly searches, competition, and top of page bid ranges are extracted for analysis.
  3. Compare Keyword Performance: The tool compares your website's keyword metrics against those of competitors, identifying strengths and areas for improvement.
  4. Generate Comparison Reports: Based on the comparison, detailed reports are created, offering insights and actionable recommendations.
  5. Save Comparison Reports: The comparison reports are saved as JSON files in a new directory named after the original Excel file within the output base directory.

Prerequisites

  • Python 3.6 or newer installed on your system.
  • Pandas library installed for data manipulation. Install via pip: pip install pandas.
  • Understanding of SEO and PPC campaigns.

Setup

Ensure the Competitor Analysis Tool script is located within your CrewAI project directory. Configure the source and output directory paths based on your project's structure.

Using the Tool

  1. Configure the source directory containing Excel files and the output directory for saving reports.
  2. Execute the script with Python, specifying any necessary parameters.
  3. Review the generated competitor analysis reports in the output directory, saved as JSON files.

Example Command

# Example command to run the Competitor Analysis Tool
python competitor_analysis_tool.py --source_directory "/path/to/excel/files" --output_base_directory "/path/to/output"

Customization

The Competitor Analysis Tool can be customized to suit various Excel file formats or to refine the comparison logic. Adjust the script as necessary to match your specific analytical needs.

Troubleshooting

If you encounter issues, ensure the Excel files are correctly formatted and the directories exist. Error messages in the console can provide insights into problems during execution.
