Content Topic Generator Usage Guide for CrewAI

This document provides detailed instructions on how to use the Content Topic Generator within the CrewAI framework. The tool generates content ideas based on keyword data, focusing on popularity, search intent, and user focus.

Workflow Overview

  1. Read Excel Files: The tool reads each Excel file in the specified source directory, extracting keyword data.
  2. Analyze Keyword Data: It analyzes keyword popularity, search intent, and user focus from the data.
  3. Generate Content Ideas: Based on the analysis, the tool generates content ideas that are likely to perform well in search engines.
  4. Save Content Ideas: The generated ideas are saved as JSON files in the specified output directory.

Prerequisites

  • Python 3.6 or newer installed on your system.
  • Access to the CrewAI framework and basic understanding of its functionality.
  • Pandas library installed for data manipulation. Install via pip: pip install pandas

Setup

Place the Content Topic Generator script within your CrewAI project directory. Configure the source and output directory paths according to your project structure.

Using the Tool

  1. Configure Directories: Set the source directory containing your Excel files and the output directory where the JSON files will be saved.
  2. Run the Script: Execute the tool using Python. It will process each Excel file to generate content ideas.
  3. Review Generated Ideas: Check the output directory for JSON files named after the original Excel files. Each file contains an array of generated content ideas.

Example Command

# Example command to run the Content Topic Generator
python content_topic_generator.py --source_directory "/path/to/excel/files" --output_base_directory "/path/to/output"

Customization

The Content Topic Generator can be customized to suit different Excel file formats or to refine the logic used for generating content ideas. Modify the script as necessary to fit your specific needs.

Troubleshooting

If you encounter any issues, ensure that the Excel files are correctly formatted and that both the source and output directories exist. Error messages in the console can provide insight into problems encountered during script execution.
